# Brand Easy – Next.js

Brand Easy signage website built with Next.js 15, Tailwind CSS 3, and TypeScript.

## Setup & Run

```bash
# 1. Install dependencies
npm install
# or
pnpm install

# 2. Start development server
npm run dev
# or
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Pages

- `/` — Home (hero, services, stats, portfolio, CTA)
- `/services` — Full services listing
- `/about` — Company story, timeline, team
- `/contact` — Enquiry form

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS 3
- **Language**: TypeScript
- **Icons**: Lucide React
- **Fonts**: Playfair Display, Cormorant Garamond, Lato (Google Fonts)

## Customise

- **Colors**: Edit `tailwind.config.ts` and the CSS variables in `app/globals.css`
- **Content**: Update component files in `components/` and page files in `app/`
- **Fonts**: Change Google Fonts import in `app/globals.css`
